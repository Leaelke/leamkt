Documentación sobre la gestión de los plugins

Aprende más: Documentación de actualizaciones automáticas

Soporte

Los plugins amplían las funcionalidades de WordPress. Una vez instalados, puedes activarlos o desactivarlos desde aquí.

La búsqueda en los plugins instalados buscará términos en su nombre, descripción o autor. Los resultados de búsqueda se actualizarán a medida que escribas.

Si quieres ver más plugins entre los que elegir, haz clic en el botón «Añadir nuevo» y podrás navegar o buscar más plugins del directorio de plugins de WordPress.org. Los plugins del directorio de WordPress.org están diseñados por terceros y son compatibles con la licencia que utiliza WordPress. Ah, y además ¡son gratis!

La mayoría de las veces, los plugins funcionan perfectamente con el núcleo de WordPress y con los otros plugins. Algunas veces, puede haber incompatibilidades entre algunos plugins, produciendo problemas. Tu sitio podría comenzar a hacer cosas raras, esto podría ser un problema. Prueba a desactivar tus plugin e ir activándolos uno a uno y comprobando que el problema no reaparezca. Es la forma de detectar el plugin problemático o la combinación de plugins problemáticos.

Si hay algún problema con un plugin y no puedes usar WordPress, borra o renombra ese archivo en el directorio /home/vol2_6/epizy.com/epiz_27614370/htdocs/wp-content/plugins y se desactivará automáticamente.

Las actualizaciones automáticas se pueden activar o desactivar para cada plugin individual. Los plugins con las actualizaciones automáticas activadas mostrarán la fecha estimada para la siguiente actualización automática. Las actualizaciones automáticas dependen del sistema de programación de tareas WP-Cron.

Las actualizaciones automáticas solo están disponibles para los plugins reconocidos por WordPress.org o que incluyan un sistema de actualización compatible.

Por favor, ten en cuenta: Los temas y plugins de terceros o el código personalizado pueden anular la programación de WordPress.

El plugin ha roto el diseño de la página u otra funcionalidad

Este tipo de problema puede solucionarse generalmente desactivando algunas opciones de WP Rocket.

Hacer clic en "Activar modo seguro" para desactivar rápidamente las opciones LazyLoad, Optimizar archivos, contenido incrustado y CDN. Luego revisa tu sitio web para ver si el problema se ha resuelto.

Modo seguro aplicado.
Revisa tu sitio web en una nueva ventana de incógnito.

¿El problema está solucionado? Ahora puedes reactivar las opciones de una en una para determinar cuál es la que causa el problema. Más información

Mi nota de PageSpeed o GTMetrix no ha mejorado

WP Rocket hace tu sitio web más rápido. Las notas de PageSpeed o GTMetrix no son un indicador de velocidad. Ni tus visitantes reales ni Google verán nunca la ”nota” de tu sitio web. La velocidad es la única métrica que importa para el posicionamiento y las conversiones.

Los de Yoast, expertos en todo lo que concierne el SEO para WordPress, afirman:

[Google] solo se fija en la velocidad de carga de tu sitio web para quién lo visita, entonces no hace falta obsesionarse sobre las notas específicas. Tienes que asegurarte que tu sitio web sea lo más rápido posible.
https://yoast.com/ask-yoast-google-page-speed/

Cómo medir el tiempo de carga de tu sitio:
https://wp-rocket.me/blog/correctly-measure-websites-page-load-time/

Por qué no deberías estar persiguiendo una nota en PageSpeed:
https://wp-rocket.me/blog/the-truth-about-google-pagespeed-insights/

No he notado ninguna diferencia en el tiempo de carga

Asegúrate de revisar tu sitio web sin estar conectado a WordPress: solo así verás las páginas en caché!

La mejor forma de verificar la mejora que proporciona WP Rocket, es realizar pruebas de velocidad. Sigue esta guía para medir correctamente el tiempo de carga de tu sitio:
https://wp-rocket.me/blog/correctly-measure-websites-page-load-time/

El plugin es demasiado complicado de configurar

Lamentamos que estés encontrando WP Rocket difícil de configurar.

WP Rocket es el único plugin de caché que aplica por defecto el 80% de las prácticas recomendadas para la optimización de la velocidad. Esto significa que no tendrás que hacer nada más que activar WP Rocket, ¡y tu sitio web será en seguida más rápido!

Las opciones adicionales no son obligatorias para obtener un sitio web más rápido: se trata de medidas de ajuste.

Para ver el beneficio que WP Rocket ya está proporcionando, mide la velocidad de tu sitio usando una herramienta como Pingdom:
https://wp-rocket.me/blog/correctly-measure-websites-page-load-time/
